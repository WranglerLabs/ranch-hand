# Ranch Hand v0.1.0-rc.15

**Classification: Public Preview.** This release is unsigned, intended for
evaluation and feedback, and is not a production-supported or GA release.

## Guided SSH host identity

Remote Linux Docker Compose setup now includes **Inspect server host key**.
Ranch Hand connects to the selected SSH endpoint with its embedded Go client,
captures the presented algorithm and SHA-256 fingerprint, and intentionally
stops before authentication so no password or private key is sent.

The operator must compare the displayed fingerprint with a trusted source such
as the Azure VM console, the server console, or the server administrator. After
confirmation, **Use verified fingerprint** writes only the public fingerprint
to the secret-free deployment plan. Changing the host or port clears the pin.

SSH passwords and private keys remain in-memory operation credentials. They
authenticate the Linux account and do not replace server-identity verification.
