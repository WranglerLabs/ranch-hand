# Ranch Hand v0.1.0-rc.24

**Classification: Public Preview.** This release is unsigned, intended for
evaluation and feedback, and is not a production-supported or GA release.

## Reliable WSL first-run onboarding

Real-mode WSL deployments now open the explicit first-run URL:
`http://127.0.0.1:8080/onboarding`. The installation success message,
already-installed action, and managed-deployment inventory all preserve that
mode-aware URL. Demo deployments continue to open the application root.

RC24 installs RepoWrangler v1.0.14. That release removes the root document from
its service-worker cache, removes the cached-root navigation fallback, deletes
older shell caches during activation, and checks for service-worker updates
without using the browser HTTP cache. Reusing WSL loopback port 8080 can no
longer resurrect an earlier Command Center shell when a new real-mode instance
is installed.
