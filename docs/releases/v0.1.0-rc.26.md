# Ranch Hand v0.1.0-rc.26

**Classification: Public Preview.** This candidate remains blocked from
publication until every source, package, public-artifact, and installation gate
listed below passes. It is unsigned and not production-supported or GA.

## First-run identity and GitHub App setup

RC26 selects RepoWrangler v1.0.16. Fresh real-mode deployments choose their
administrator identity provider before connecting an estate. GitHub identity
and Microsoft Entra ID are supported, and initial setup remains available until
the first verified allowlisted administrator signs in.

Loopback and private-network GitHub App manifests omit webhook configuration,
because GitHub rejects hooks that are not reachable on public HTTPS. These
deployments use scheduled and manual synchronization. Public HTTPS targets keep
their webhook subscriptions.

## Remote Linux setup

Remote real-mode installation generates a one-time setup token, sends it only
in the protected runtime environment, and displays it with the exact private-LAN
URL in the active Ranch Hand result. The token is not written to the deployment
plan or lifecycle journal.
