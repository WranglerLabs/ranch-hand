# Ranch Hand v0.1.0-rc.13 — Public Preview

Released 2026-07-18. This unsigned preview removes the private-registry
dependency from the local WSL Docker Compose installation path.

## What changed

- Ranch Hand downloads the exact RepoWrangler v1.0.10 Linux AMD64 image archive
  from this immutable public release instead of asking WSL Docker to pull from
  GHCR.
- The archive is accepted only when its size is exactly 286,575,554 bytes and
  its SHA-256 is
  `bc2c7507b592a6da58ec1eeed199d2c3b028bdb6a6b73f22a00ff7aab46ada5e`.
- Ranch Hand caches the verified archive under the current Windows user and
  streams it into Docker Engine inside the selected WSL distribution.
- Compose selects `repo-wrangler-server:v1.0.10-ranch-hand` and enforces
  `pull_policy: never`. The WSL path does not make a registry request and does
  not require a GitHub account, GitHub token, or Docker registry login.
- The runtime image ID, original immutable release digest, generated override,
  ownership marker, Docker labels, health, and v1.0.10 release identity remain
  verified.

## Verified archive

- Asset: `repo-wrangler-v1.0.10-linux-amd64-image.tar.gz`
- Size: 286,575,554 bytes
- SHA-256:
  `bc2c7507b592a6da58ec1eeed199d2c3b028bdb6a6b73f22a00ff7aab46ada5e`
- Loaded image ID:
  `sha256:89d1b4091137eef57c91270d363fb6c76e6d60c94dcac92b129b2b8629f45093`

## Scope

This correction applies to **Local Docker Compose — WSL**. Remote Linux Compose
still uses the target Docker client's configured registry access and remains a
separate Preview adapter.

## Classification

Public Preview. The executable is unsigned and may trigger Windows reputation
warnings. It is intended for evaluation and feedback, not production support.
