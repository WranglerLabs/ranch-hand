# Ranch Hand v0.1.0-rc.18

**Classification: Public Preview.** This release is unsigned, intended for
evaluation and feedback, and is not a production-supported or GA release.

## Visible Remote Linux installation progress

Clicking **Install RepoWrangler** now immediately reports that the operation was
submitted. While the blocking SSH and Compose work runs, the panel refreshes
the durable lifecycle phase every second and tells the operator to keep Ranch
Hand open.

Completion and errors appear directly beneath the install button. A missing
prepared plan, unchecked confirmation, or expired in-memory SSH credential now
produces a specific recovery instruction instead of returning silently.
