# Ranch Hand v0.1.0-rc.5 — Public Preview

Released 2026-07-17. This Preview fixes silent WSL install feedback and recovery.

## Fixed

- Selecting **Install in WSL** without accepting the confirmation now displays
  the exact required action rather than silently ignoring the click.
- A submitted install immediately reports that it is running and polls the
  durable lifecycle journal every second. Long image pulls show the current
  phase and tell the operator to keep Ranch Hand open.
- Errors remain visible beside the install action as well as in the plan result.
- Interrupted WSL operations can be safely closed or recovered without Ranch
  Hand incorrectly requiring an SSH or cloud credential.

The executable remains unsigned and is not GA or production-supported.
