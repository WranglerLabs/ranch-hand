# Ranch Hand v0.1.0-rc.4 — Public Preview

Released 2026-07-17. This Preview corrects Compose collision handling and reduces
remote Linux setup input.

## Fixed

- Local WSL Compose now defaults to project `repo-wrangler-ranch-hand`, avoiding
  collision with an existing ordinary `repo-wrangler` Compose project.
- A genuine WSL collision is reported as local WSL state and is never described
  as a remote Compose deployment. Ranch Hand continues to refuse destructive
  replacement of unknown containers, volumes, or installation directories.
- Remote Linux Docker Compose now prepopulates SSH port `22` and project
  `repo-wrangler-ranch-hand`.
- Entering a remote SSH username fills `/home/<user>/.repo-wrangler-ranch-hand`,
  or `/root/.repo-wrangler-ranch-hand` for root. A customized directory is never
  overwritten when the username changes.

## Preview boundary

The executable remains unsigned and is not GA or production-supported. WSL and
remote Linux currently support new evaluation installs; their complete
lifecycle remains a GA gate.
