# Ranch Hand v0.1.0-rc.12 — Public Preview

Released 2026-07-18. This unsigned preview fixes the Compose image-pull failure
boundary for local WSL and remote Linux targets.

## What changed

- Ranch Hand pulls the exact digest-pinned RepoWrangler image before creating
  the target installation directory or any ownership state.
- The pull uses the target Docker client's configured registry credentials.
- A registry authorization or availability failure returns directly without
  leaving an installation directory, marker, container, volume, or recovery
  lock.
- After the verified pull succeeds, Compose starts with `--pull never`, so it
  cannot repeat the registry request after mutation begins.

## Classification

Public Preview. The executable is unsigned and may trigger Windows reputation
warnings. It is intended for evaluation and feedback, not production support.
