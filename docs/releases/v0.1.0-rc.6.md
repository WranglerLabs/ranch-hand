# Ranch Hand v0.1.0-rc.6 — Public Preview

Released 2026-07-17. This Preview fixes interrupted WSL install recognition.

## Fixed

- WSL preflight now correlates an existing installation directory with Ranch
  Hand's durable lifecycle records.
- An interrupted owned install displays **Recover interrupted WSL installation**
  directly beside the blocked preflight. Recovery still requires exact marker,
  transferred-file hash, and Docker-label ownership evidence.
- A committed installation is identified as already installed and links to its
  loopback endpoint instead of offering a duplicate install.
- An unknown or unowned directory remains blocked and untouched.

The executable remains unsigned and is not GA or production-supported.
