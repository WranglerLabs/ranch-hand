# Ranch Hand v0.1.0-rc.7 — Public Preview

Released 2026-07-17. This Preview completes the earliest interrupted WSL install
recovery boundary.

## Fixed

- When Ranch Hand stopped after creating its dedicated WSL directory but before
  writing the ownership marker, recovery can remove that exact directory only
  if it is still empty.
- A markerless directory containing any file or subdirectory remains blocked
  and untouched. Recovery does not recursively delete or adopt it.

This supersedes rc.6 for WSL evaluation testing. The executable remains
unsigned and is not GA or production-supported.
