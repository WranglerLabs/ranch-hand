# Ranch Hand v0.1.0-rc.8 — Public Preview

Released 2026-07-17. This Preview fixes partial WSL cleanup, stale bound plans,
and deployment visibility.

## Fixed

- **Remove failed installation and release lock** cleans a partial pre-marker
  transfer only when it contains Ranch Hand's fixed filenames and no Compose
  container, volume, symlink, invalid marker, or unknown content.
- Editing a project name or other deployment input discards the old bound plan,
  preflight, staged bundle, and confirmation. A newly created plan has its own
  deployment identity and cannot accidentally submit the old project's lock.

## Added

- **Managed deployments** is the authoritative Ranch Hand inventory for target,
  project, version, state, update time, and local WSL endpoint. The launcher
  terminal remains only a startup and diagnostic surface.

This supersedes rc.7 for WSL evaluation testing. The executable remains
unsigned and is not GA or production-supported.
