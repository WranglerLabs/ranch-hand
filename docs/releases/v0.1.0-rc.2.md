# Ranch Hand v0.1.0-rc.2 — Public Preview

Released 2026-07-17. This Preview corrects the release-selection workflow found
while exercising the first public build.

## Changed

- **Latest stable (recommended)** is the default release choice.
- Ranch Hand discovers the newest published stable RepoWrangler release that
  contains a valid manifest and artifact for the selected target.
- **Latest prerelease** is an explicit opt-in choice.
- **Specific version (advanced)** preserves immutable version pinning and
  rollback-oriented workflows.
- Release discovery is performed by the loopback backend; the selected release
  still passes the full manifest, artifact, provenance, SBOM, size, and digest
  verification before it can be used.

## Preview boundary

This executable is unsigned and is not a GA or production-supported release.
See [GA readiness](../ga-readiness.md) for the complete promotion contract.
