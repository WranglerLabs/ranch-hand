# Ranch Hand v0.1.0-rc.28

**Classification: Public Preview.** Unsigned, not production-supported, and
not generally available.

## Dynamic RepoWrangler release catalog

RC28 removes the Ranch Hand build-time ceiling on compatible RepoWrangler
releases. The local interface fetches the current catalog for the selected
target, lists every compatible stable or preview release, and provides an
explicit **Fetch available releases** action. Operators can still enter an
exact immutable version through the advanced path.

Catalog entries establish availability only. Ranch Hand still verifies the
selected release manifest, deployment bundle, SPDX SBOM, Sigstore provenance,
declared byte counts, and SHA-256 digests before creating a plan or mutating a
target.

## Dynamic WSL and Remote Linux images

For WSL and Remote Linux, later RepoWrangler releases no longer require a new
hard-coded companion record in Ranch Hand. Ranch Hand downloads the official
versioned Linux image archive, restricts redirects and response size, verifies
the archive digest against the release's Sigstore provenance, and inspects the
archive for the exact versioned tag and immutable image identities before
loading it. Existing explicit records remain supported for legacy releases.

New ownership markers record the exact verified runtime image ID. Post-install
verification compares the running container with that recorded identity. WSL
also preserves the public plan's deployment ID when it uses the shared internal
Remote Linux Compose transport, keeping apply, verify, recovery, and uninstall
on the same ownership boundary.
