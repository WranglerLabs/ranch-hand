# Ranch Hand v0.1.0-rc.9 — Public Preview

Released 2026-07-17. This Preview corrects the WSL/SSH transfer defect that
created empty deployment files.

## Fixed

- File payloads now reach the complete atomic remote write command. Compose,
  environment, and ownership-marker files are no longer empty.
- **Remove failed installation and release lock** recognizes and safely removes
  the exact empty files created by rc.8 and earlier candidates.
- Same-tab refresh retains the authenticated session without putting the token
  back in the address bar.
- Compose failures include bounded command output instead of only an exit code.

## Preflight

Ranch Hand now separately reports a missing Docker command, an unavailable or
unauthorized Docker Engine, a missing Compose v2 plugin, and an occupied Windows
loopback port 8080. It does not install privileged Docker prerequisites.

This supersedes rc.8 for all WSL and remote Linux testing. The executable remains
unsigned and is not GA or production-supported.
