# Ranch Hand v0.1.0-rc.27

**Classification: Public Preview.** Unsigned, not production-supported, and
not generally available.

## WSL real-mode installation

RC27 fixes the RC26 regression that normalized a local WSL deployment through
the Remote Linux Compose implementation and then incorrectly required Remote
Linux's 32–256 character setup token. Local WSL remains protected by loopback
and now installs real mode with generated session and encryption secrets but no
remote handoff token.

The exact corrected Windows candidate completed a clean WSL lifecycle:

- verified and staged the immutable RepoWrangler v1.0.16 release;
- passed live WSL target preflight;
- applied, passed readiness and exact-version verification, and committed;
- reported first-run mode with no administrator identity selected;
- generated a loopback GitHub App manifest without webhook fields; and
- completed managed uninstall with no project containers or volumes remaining.

## Lifecycle and image handling

Managed permanent uninstall covers Docker Desktop, WSL Compose, Remote Linux
Compose, Azure Container Apps, and Cloudflare Worker plus D1 with exact
ownership checks. Docker Desktop loads the verified public release archive
through the Docker Engine API, and Azure rejects a private or digest-mismatched
runtime image before creating resources.

## Verification gates

- Windows Go tests, vet, vulnerability scan, web typecheck/build, release
  contracts, and executable identity smoke.
- Linux Go tests and vet, preventing platform-specific WSL adapter drift.
- Public release manifest, archive, OCI identity, anonymous registry, Compose
  start, and RepoWrangler first-run smoke.
- Go and JavaScript/TypeScript CodeQL analyses.
