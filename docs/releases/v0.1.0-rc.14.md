# Ranch Hand v0.1.0-rc.14 — Public Preview

Released 2026-07-18. This unsigned preview stops silently forcing demo mode for
local WSL Docker Compose installations.

## What changed

- The WSL target now has an explicit **Demo mode** toggle in the secret-free
  deployment plan.
- Demo mode defaults to **off**. Ranch Hand starts RepoWrangler with
  `DEMO_MODE=false`, generates unique 256-bit session and
  credential-encryption secrets, writes them only to the protected target
  environment, and opens RepoWrangler's real first-run provider setup.
- Turning demo mode on intentionally selects the existing mock-data profile.
- The install confirmation repeats the selected mode before Ranch Hand creates
  the deployment.
- Existing rc.13-and-earlier plans without the field retain their original demo
  interpretation for compatibility and safe recovery.

## WSL image delivery

The rc.13 correction remains in effect: WSL downloads and verifies the public
RepoWrangler v1.0.10 image archive, loads it directly into WSL Docker Engine,
and runs Compose with registry pulls disabled. No GitHub account, token, or
registry login is required.

## Verification

The v1.0.10 image was started with Ranch Hand's real-mode environment.
`/health/ready` returned `demoMode: false`, `/auth/config` returned
`setupMode: true`, and the test deployment was removed.

## Classification

Public Preview. The executable is unsigned and may trigger Windows reputation
warnings. It is not yet generally available or production-supported.
