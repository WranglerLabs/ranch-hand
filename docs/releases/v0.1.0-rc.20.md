# Ranch Hand v0.1.0-rc.20

**Classification: Public Preview.** This release is unsigned, intended for
evaluation and feedback, and is not a production-supported or GA release.

## Remote Linux is reachable after installation

New Remote Linux plans expose RepoWrangler on the server's private IPv4 address
at port 8080. Ranch Hand still manages the deployment through SSH, but it now
verifies the page from the Windows control workstation and shows the exact
clickable URL after the lifecycle journal commits. Public-IP targets remain
blocked because this evaluation endpoint uses plain HTTP.

## WSL black-page recovery

This candidate installs RepoWrangler v1.0.12. Its web startup path replaces the
failed cached asset set that could leave a healthy WSL deployment showing only
a black page. Ranch Hand additionally verifies the HTML shell and referenced
JavaScript asset before it reports WSL or Remote Linux installation success.

## Manual removal runbook

The [removal runbook](../uninstall.md) documents ownership checks and separate
retain-data and permanent-delete procedures for every supported target:
Docker Desktop, WSL, Remote Linux, Azure Container Apps, and Cloudflare. It also
documents removal of the portable Ranch Hand executable. Automated uninstall
and catalog reconciliation remain follow-on lifecycle work.
