# Ranch Hand v0.1.0-rc.19

**Classification: Public Preview.** This release is unsigned, intended for
evaluation and feedback, and is not a production-supported or GA release.

## Remote Linux installation without registry access

Remote Linux Docker Compose now uses the same verified public RepoWrangler
image archive as the WSL target. Ranch Hand downloads the archive to its
Windows cache, verifies the declared size and SHA-256, and streams the binary
archive directly to `docker image load` over the authenticated, host-key-pinned
SSH connection.

Before creating the installation directory or Compose project, Ranch Hand
inspects the loaded local tag and requires the exact expected Docker image ID.
The generated Compose override selects that loaded image and sets
`pull_policy: never`. The target therefore requires no GitHub account, GHCR
login, registry token, or direct registry access.

## Root cause corrected

The prior Remote Linux apply path ran `docker image pull` after a successful
SSH, Docker, Compose, directory, and project preflight. Because the referenced
GHCR package did not allow that anonymous pull, installation failed with
`unauthorized` even though preflight reported ready. The apply path no longer
makes that registry request.
