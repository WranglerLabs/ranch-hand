# Ranch Hand v0.1.0-rc.17

**Classification: Public Preview.** This release is unsigned, intended for
evaluation and feedback, and is not a production-supported or GA release.

## Guided Docker prerequisites

Docker-backed targets no longer stop at a missing Engine or Compose check.
Preflight remains read-only and offers a separate confirmed recovery action:

- WSL Ubuntu/Debian installs Docker Engine and Compose v2 through the local WSL
  root boundary, starts Docker, grants the selected WSL user access, and reruns
  preflight.
- Remote Ubuntu/Debian performs the same bounded operation through the verified
  SSH connection using root or sudo. Passwords remain in memory.
- Docker Desktop installs through Windows Package Manager. Administrator
  approval, first-run terms, or starting Docker Desktop may still be required
  before the native Engine preflight becomes ready.

Unsupported Linux distributions are rejected without executing an unknown
package manager. Ranch Hand never installs Docker during read-only preflight.
