# Ranch Hand v0.1.0-rc.23

**Classification: Public Preview.** This release is unsigned, intended for
evaluation and feedback, and is not a production-supported or GA release.

## Delete an existing WSL deployment in Ranch Hand

Every active local WSL Compose installation now appears under **Managed
deployments** with a **Permanently remove deployment** action. The action
requires an explicit acknowledgement that RepoWrangler data will be deleted.

Ranch Hand then verifies the recorded deployment identity, exact target-side
ownership marker, transferred-file hashes, and Docker container and volume
labels before running Compose removal with volumes. It removes the dedicated
installation directory only after those checks pass and commits the durable
lifecycle inventory record as uninstalled. Unknown or changed resources remain
blocked and untouched.

If removal is interrupted, Ranch Hand retains the operation lock and exposes
the existing recovery action. Retrying is idempotent and succeeds only when the
owned resources and directory are absent.

Retain-data uninstall and managed uninstall for other targets remain future
lifecycle work; their ownership-checked manual procedures remain documented.
