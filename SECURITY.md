# Security policy

## Supported versions

Ranch Hand has not issued a generally available release. Unsigned release candidates are evaluation artifacts, not supported production releases. Security fixes will apply to the newest published release once trusted releases begin.

## Reporting a vulnerability

Do not open a public issue for a suspected vulnerability. Use GitHub's **Report a vulnerability** link on the repository Security tab to submit a private report. Include affected versions, reproduction steps, impact, and any suggested mitigation.

Do not include live credentials or personal deployment data. Maintainers will acknowledge a complete report as capacity permits; this is a community-maintained open-source project and does not include a commercial response-time SLA.
