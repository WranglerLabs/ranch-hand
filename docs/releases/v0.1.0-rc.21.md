# Ranch Hand v0.1.0-rc.21

**Classification: Public Preview.** This release is unsigned, intended for
evaluation and feedback, and is not a production-supported or GA release.

## WSL image identity compatibility

RC20 could download and load the correct verified RepoWrangler v1.0.12 image
archive, then reject it because Docker image stores do not all expose the same
digest representation through `docker image inspect`. Depending on the Docker
Engine and image-store backend, the loaded identity can be the verified
registry index, platform manifest, or image config digest.

RC21 pins all three identities from the immutable release archive and accepts
only those values. Unknown image identities remain rejected. The same check is
used for WSL and Remote Linux image loading and runtime verification.
