# Ranch Hand v0.1.0-rc.22

**Classification: Public Preview.** This release is unsigned, intended for
evaluation and feedback, and is not a production-supported or GA release.

## Fresh WSL setup opens onboarding

RC22 installs RepoWrangler v1.0.13. A fresh real-mode deployment now waits for
the public setup-state response before mounting protected dashboard pages. If a
request nevertheless reaches the sign-in route during initial boot, the route
recognizes setup mode and opens `/onboarding` instead of showing a sign-in page
with no configured provider.

## Immutable image trust

Ranch Hand pins the v1.0.13 registry index, platform manifest, image config,
offline archive SHA-256, and exact archive size. Unknown Docker image identities
remain rejected.
