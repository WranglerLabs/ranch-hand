# Ranch Hand v0.1.0-rc.25

**Classification: Public Preview.** This release is unsigned, intended for
evaluation and feedback, and is not a production-supported or GA release.

## GitHub App setup across every target

RC25 installs RepoWrangler v1.0.15. The onboarding wizard now posts the public
GitHub App manifest directly to GitHub for either a personal account or an
organization. It no longer opens an intermediate local `/setup/github-app`
page. The manifest keeps webhook, setup callback, and OAuth callback URLs on
the active deployment origin.

The same v1.0.15 source release produces Compose, Azure Container Apps, and
Cloudflare bundles. Node/Compose dispatch and Worker setup routes have separate
regression coverage.
